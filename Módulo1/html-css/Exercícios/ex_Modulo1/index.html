Estilo Locais / internos

Vantagens: Praticidade de evitar repetição de códigos, altera todos, menos trabalhosa, utilizado bastente no dia a dia para um documento.

Desvantagens: estilos ocupam muitas linhas de código e não é utilizado por outras páginas

        
            <head>
            ...
                <style>
                    body {            (tudo que colocar aqui dentro vai ser aplicado no body
                        background-color: greenyellow;
                        font-family: Arial,Helvetica, sans-serif;
                        font-size: 20px;
                    }   
                    h1 {               (tudo que colocar aqui dentro vai ser aplicado nos h1
                        color: blue; 
                        background-color: blanchedalmond; 
                    } 
                    h2 {               (tudo que colocar aqui dentro vai ser aplicado nos h2
                        color: darkred;
                    } 
                    p {
                        text-align: jusify;
                    }
                </style>
            </head>

Estilo Externos - style em outro arquivo

Vantagens: Praticidade de evitar repetição de códigos, altera todos, menos trabalhosa, utilizado bastente no dia a dia para para vários documento.

        
            <head>
            ...
                <link rel="stylesheet" href="style.css">   (link para acessar o documento com os estilos para página)
            </head>
